MVP4Log

POC per Innovathon2016!

Log digger

Console per l'estrazione di log in dati strutturati. Si compone di un server HTTP con API per configurare metriche di analisi dei log e per esportare i costrutti risultanti. La visualizzazione é una pagina (AngularJS). Il server dialoga con un nodo elasticsearch. Gli indici indirizzano i files di log raccolti sul file system.

Scopo del progetto

Log digger é una POC per la presentazione di un nuovo processo di sviluppo in Azienda.
